# 🎬 News Shorts Live

A browser-based YouTube Shorts news generator that fetches live headlines and exports vertical 9:16 video previews.

No backend required.  
Runs fully in the browser.

---

## 🚀 Features

- 🔴 Live news fetching (AI / Finance / Geopolitics)
- 🎨 Theme customization (Accent + Background colors)
- 📰 Auto ticker + category tags
- 🔊 Breaking news sound effect
- 📡 Visualizer animation
- 📱 9:16 YouTube Shorts preview
- 🎥 WebM video export (6 seconds)

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- html2canvas
- MediaRecorder API
- GNews API (free tier)
